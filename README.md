# Fraukes Kurse

Statische Kursseite (HTML/CSS/JS). Notizen/Checklisten speichern lokal im Browser (localStorage).

## GitHub Pages
1. Repo erstellen (public oder private).
2. Dateien hochladen (Ordnerstruktur beibehalten).
3. GitHub: Settings → Pages
4. "Build and deployment" → Source: Deploy from a branch
5. Branch: main, Folder: / (root)
6. Speichern → URL erscheint nach kurzer Zeit.

## Lokal öffnen
Einfach `index.html` doppelklicken.
